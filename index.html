Sign in with GitHub to access your construction diary. One-time setup takes 3 minutes.

One-time GitHub OAuth setup

1
Go to github.com/settings/developers → OAuth Apps → New OAuth App
2
Fill in: Application name: Site Diary  |  Homepage URL: https://dylashsav-beep.github.io/site-diary/  |  Callback URL: https://dylashsav-beep.github.io/site-diary/
3
Click Register application. Then click Generate a new client secret. Copy both the Client ID and Client Secret.
4
Paste them below and click Sign In. GitHub will ask you to authorise the app once.
Note: Because GitHub OAuth requires a server for the secret exchange, we use Netlify Functions as a free middleman — just for the login step. Your data still lives entirely in your GitHub repo. See the setup guide for the 5-minute Netlify step.
