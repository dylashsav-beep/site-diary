Site Diary — SharePoint Setup

Before you sign in, you need to register this app in Azure AD and create two SharePoint lists. This is a one-time setup. After that, paste your App (Client) ID below and click Sign In.

One-Time Setup Steps

1
Go to portal.azure.com → Azure Active Directory → App Registrations → New Registration. Name it Site Diary. Set Redirect URI to Single-page application (SPA) and paste the URL where this HTML file is hosted.
2
In the app registration, go to API Permissions → Add → Microsoft Graph → Delegated → add Sites.ReadWrite.All. Then grant admin consent.
3
Go to your SharePoint site → Site Contents → New → List. Create a list named exactly SiteDiaryJobs. Add columns: JobNumber (text), Client (text), LRate (number), PRate (number), SRate (number), Resources (multiple lines of text).
4
Create a second list named exactly SiteDiaryEntries. Add columns: JobId (number), DiaryDate (date), Weather (text), Activities (multiple lines), Hours (multiple lines), GeneralNotes (multiple lines), Deliveries (multiple lines), Inventory (multiple lines).
5
Copy the Application (Client) ID from your app registration overview and paste it below.
